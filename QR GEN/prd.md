# Product Requirements Document (PRD): WiFi QR Code Generator

## 1. 개요 (Overview)
본 문서는 사용자가 와이파이 이름(SSID)과 비밀번호를 입력하면, 스마트폰 카메라 등으로 스캔하여 해당 와이파이에 바로 접속할 수 있는 QR 코드를 생성해주는 웹 애플리케이션의 요구사항을 정의합니다.

## 2. 기술 스택 (Tech Stack)
- **프레임워크**: Next.js (App Router 권장)
- **스타일링**: Tailwind CSS
- **QR 코드 생성 라이브러리**: `qrcode.react` (또는 유사 라이브러리)
- **언어**: TypeScript (권장) / JavaScript

## 3. 핵심 기능 (Core Features)

### 3.1. 사용자 입력 폼 (Input Form)
- **SSID (와이파이 이름) 입력 필드**: 텍스트 입력. (필수)
- **비밀번호 입력 필드**: 텍스트 입력. 비밀번호 숨김/보임 토글 기능 포함. (선택: 공개 와이파이의 경우 비밀번호가 없을 수 있음)
- **암호화 및 네트워크 타입 (선택 사항)**: WPA/WPA2/WPA3 (기본값), WEP, 없음. (가장 널리 쓰이는 WPA 계열을 기본으로 설정하여 사용자 편의성 증대)
- **생성 버튼**: 입력된 폼 데이터를 기반으로 QR 코드를 렌더링하는 엔트리 포인트.

### 3.2. QR 코드 렌더링
- 와이파이 접속을 위한 범용 QR 코드 포맷(`WIFI:T:WPA;S:네트워크이름;P:비밀번호;;`)에 맞추어 문자열을 조합합니다.
- 조합된 문자열을 QR 코드 라이브러리를 통해 시각적인 데이타 매트릭스(QR Code)로 화면에 실시간 또는 버튼 클릭 시 표시합니다.

### 3.3. 인터페이스 (UI/UX)
- **반응형 디자인 (Responsive Web)**: 모바일 단말기 및 데스크톱 환경 모두에서 최적의 레이아웃을 제공합니다. (Tailwind CSS 활용)
- **직관적인 레이아웃**: 입력 영역과 결과(QR 코드) 영역을 깔끔하게 분리하여 사용자가 혼란 없이 사용할 수 있도록 구성합니다.

## 4. 부가 기능 (Optional)
- **다운로드 기능**: 생성된 QR 코드를 이미지(PNG 등)로 다운로드하여 인쇄나 공유가 가능하도록 지원.
- **디자인 커스터마이징**: QR 코드 색상 변경 기능.

## 5. 다음 개발 단계 (Next Steps)
1. **Next.js 프로젝트 초기화**: `npx create-next-app`을 이용해 프로젝트 세팅 (Tailwind CSS 포함).
2. **초기 UI 구성**: 입력 폼 및 화면 레이아웃 스케치 및 구현.
3. **상태 관리 및 로직 연동**: React 상태(State)를 사용하여 사용자 입력을 관리하고 QR 코드 컴포넌트에 전달.
4. **테스트 및 디버깅**: 실제 모바일 기기의 카메라 앱으로 QR 코드를 스캔하여 정상 접속되는지 확인.
